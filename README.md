# SoussMassaMonitor

لوحة لرصد الأخبار السياسية والانتخابية المتعلقة بجهة سوس ماسة من قائمة محددة من المصادر المغربية العربية والفرنسية.

تُحدَّث البيانات ولوحة HTML آلياً كل ست ساعات بواسطة GitHub Actions.

## الملفات الأساسية

- `run_pipeline.py`: الجمع والتصنيف وبناء لوحة التحكم.
- `config/`: المصادر والقواميس وقواعد التصنيف.
- `data/news.json`: قاعدة الأخبار المقبولة.
- `output/index.html`: لوحة التحكم المنشورة.
